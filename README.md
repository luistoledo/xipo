### tema 1: ver jugar, jugar sin jugar

me interesa el acto de jugar a través de leer sobre un juego

en concreto recuerdo de niño leer una guía sobre _Gabriel Knight: Sins of the Fathers_ en una revista de videojuegos y quedar prendado, el artículo narraba la run del juego como una historia. fue mi única forma de acceder al juego durante muchos años. para esta sesión escribí un pequeño [ensayo](1-jugar_sin_jugar/ensayo.md) reflexionando sobre las guías de juego como literatura ergódica, y prototipé un generador de guías de juego sobre un juego inexistente _pokemon infrarrojo_

una guía de pokemon rojo (encontrada en línea) alimenta un generador de texto semialeatorio basado en cadenas de markov, que reemplaza ciertas palabras clave con opciones ofrecidas al jugador-lector

el prototipo se presenta como un falso escritorio de ordenador, con un único archivo _poke-ir.txt_ que cada vez que se abre, genera la guía de juego para alguna ruta aleatoria

se quedan muchas ideas que seguir explorando, como una regeneración dinámica de texto según las elecciones del lector, persistir los textos generados por cada usuario a suerte de coparticipación, etc
